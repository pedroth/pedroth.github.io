LinesSurfaces is a simple application that let you see parametrized surfaces and curves in 3 dimensions. These are surfaces given by a function $\mathbf{f} : (u,v)\in \mathbb{R^2} \rightarrow (f_1(u,v), f_2(u,v), f_3(u,v))$ and curves given by a function $\boldsymbol{\gamma}:t\in\mathbb{R}\rightarrow(\gamma_1(t), \gamma_2(t), \gamma_3(t))$.
After setting up your curve or surface you can generate .obj file of the surface/curve to be used in other programs (press g).
Press "h" for help. Feel free to download the program below.
