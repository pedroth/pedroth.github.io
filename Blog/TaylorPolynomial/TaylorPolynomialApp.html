Taylor Polynomial

Last time we saw the fundamental theorem of calculus, let have a function $f:x\rightarrow y$ where $x \in \mathbb{R}$ and $y \in \mathbb{R}$, then $\int_{a}^{b} f'(x) dx = f(b) - f(a)$. This is nice because I can get a function from its derivative like this: $$f(x) = f(x_{0}) + \int_{x_{0}}^{x} f'(\tau) d\tau \;\;\;\;\;\;\;\;(1)$$ But if you recall: $$\lim_{h\rightarrow0} \; \sum_{i= 0}^{n}f(a+ih)h = \int_{a}^{b} f(x) dx, n= \left \lfloor \frac{b-a}{h} \right \rfloor$$ and this is a really difficult to compute, well actually is not that difficult ,but maybe there is a simpler method. Imagine that you have all the derivatives of a function at every point, you have got $ f'(x) ,\; f''(x), \; f^{(3)}(x), \; ...$. So we could also do: $$f'(x) = f'(x_{0}) + \int_{x_{0}}^{x} f''(\tau) d\tau\;\;\;\;\;\;\;\;(2)$$ From (1) and (2) we get: $$f(x) = f(x_{0}) + \int_{x_{0}}^{x}\left [ f'(x_{0}) + \int_{x_0}^{\tau}f''(u)du \right ]d\tau$$ $$\Leftrightarrow f(x) = f(x_{0}) + \int_{x_{0}}^{x} f'(x_{0}) + \int_{x_{0}}^{x}\int_{x_0}^{\tau}f''(u)du d\tau $$ Note that $\int_{x_{0}}^{x} f'(x_{0})$ is the area of a constant function, hence: $$\Leftrightarrow f(x) = f(x_{0}) + f'(x_{0})(x-x_{0}) +\int_{x_{0}}^{x}\int_{x_0}^{\tau}f''(u)du d\tau$$ We can also expand $f''(x) = f''(x_{0}) + \int_{x_{0}}^{x}f^{(3)}(\tau)$ and get: $$f(x) = f(x_{0}) + f'(x_{0})(x-x_{0}) + \int_{x_{0}}^{x}\int_{x_0}^{\tau}f''(x_{0})du d\tau + \int_{x_{0}}^{x}\int_{x_0}^{\tau}\int_{x_{0}}^{u} f^{(3)}(v) dvdud\tau$$ We can simplify $\int_{x_{0}}^{x}\int_{x_0}^{\tau}f''(x_{0})du d\tau = \int_{x_{0}}^{x}f''(x_{0})(\tau - x_{0}) d\tau = \frac{f''(x_{0})(x-x_{0})^{2}}{2}$ and we get: $$f(x) = f(x_{0}) + f'(x_{0})(x-x_{0}) +\frac{f''(x_{0})(x-x_{0})^{2}}{2} + \int_{x_{0}}^{x}\int_{x_0}^{\tau}\int_{x_{0}}^{u} f^{(3)}(v) dvdud\tau$$ If we repeat the same procedure for all derivatives(infinite) we will get the taylor expansion : $$f(x) = f(x_{0}) + f'(x_{0})(x-x_{0}) +\frac{f''(x_{0})(x-x_{0})^{2}}{2} + \frac{f^{(3)}(x_{0})(x-x_{0})^{3}}{3!} + \dots $$ $$f(x) = \sum_{n=0}^{\infty}\frac{f^{(n)}(x_{0})}{n!}(x-x_{0})^{n} \;\;\;\;\;\;\;\;(3)$$ And finally an application, lets say we wanted to find a function $y$ such that $y' = y$ and $y(0) =1$. If you think a little we will find that $y^{(i)} = y , i > 0$, then by using (3) and the fact $y(0) =1$ we know $y^{(i)}(0)= 1, i \geq 0$, therefore: $$y(x) = \sum_{n=0}^{\infty}\frac{1}{n!}(x-0)^{n} $$ This function is the well known $y = e^{x}$
